# Code Path Mortality Tracker

## Overview
The Code Path Mortality Tracker is a simple C program that simulates software reliability by monitoring different code paths. It records executions and failures, then calculates a mortality rate to indicate how frequently a path fails.

## Features
- Track execution of multiple code paths
- Record success and failure counts
- Calculate mortality rate
- Log events to a file

## Technologies
- C Programming Language
- Standard C Libraries
- File-based logging
